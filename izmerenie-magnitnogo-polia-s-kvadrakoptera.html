Измерение магнитного поля с квадракоптера

Мой небольшой проект по работе с геоданными

"Мы абсолютно не геологи и в душе не представляем насколько эти значения меняются со временем и с местом"

-- Одна из вводных
Минималистичный экран первой версии

Небольшая программа написанная на Electron.js с использованием API Яндекс.Карт для местной компании, работающей с данными по геомагнитному полю земли.
Вариант с более размытой тепловой картой

Особенности проекта

  • поиск постоянных источников отклонения значений
  • фильтрация и отброс аномальных значений
  • Построение тепловой карты значений опираясь на точки в которых данные были собраны
Измерения проводились от 1 до нескольких домов, до десятков на десять квадратных метров. Пример зашумленного измерения

Инструментарий

  • Electron.js
  • Яндекс-карты
  • OpenStreet Maps
  • pandas
  • python
